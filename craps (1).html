System Rules Reference
  • Passline: $100 on OFF (come-out). Wins 7/11, loses 2/3/12. Point hit = win 1:1, 7-out = lose.
  • Don't Come: $200 placed on first ON roll after point. Wins 2/3, pushes 12, loses 7/11. Number → Lay (wins on 7, loses if number hit).
  • Come: $25 on every roll after DC placed. Wins 7/11, loses 2/3/12. Number → Place (wins if number hit, loses on 7).
  • DC Progression: $200 → $300 only if NOT in overall profit. Resets to $200 when in profit.
  • Come Progression: $25 → $50 → $75… (+$25 per loss) only if NOT in overall profit. Resets to $25 when in profit.
  • Lay/Place on OFF: Toggle whether existing lay/place bets are active during come-out rolls.
